# WGS Taxonomy Draft v1 — Full Review Document

**How to review:** for each topic, edit/cut/add seed phrases directly in this doc (track changes however's easiest for you — strikethrough, comments, rewrites inline, whatever). Paste back whatever you've changed, or the whole doc with edits, and I'll fold it in.

**Reminder of what each field means:**
- **Topic + Category** = one pair = one row in the eventual taxonomy file. Same topic name can repeat under different categories — each is a separate, independently-authored pair.
- **Seed phrases** = short raw ideas (not full sentences of copy) that the AI expands into full posts at generation time. Aim is a mix: some are a fact/insight/mistake to explore, some are a pure feeling/truth to sit with — no argument, no fix, just something true.
- **Fact-check** = yes/no — flags whether this pair's phrasing includes any factual/statistical/medical claim requiring a real grounding source (`requires_citation: true` downstream).
